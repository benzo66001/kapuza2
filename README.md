## Описание программы

**Random Task Generator** — это GUI‑приложение для генерации случайных задач, которое помогает разнообразить свой день и не забывать о важных делах. Программа позволяет:

- ✅ Генерировать случайные задачи из разных категорий
- ✅ Просматривать историю всех сгенерированных задач
- ✅ Добавлять свои собственные задачи
- ✅ Фильтровать историю по категориям
- ✅ Удалять задачи из истории
- ✅ Сохранять всё в JSON-файл
- ✅ Загружать историю при запуске

Приложение разработано на Python с использованием библиотеки `tkinter` для графического интерфейса. Все данные сохраняются локально в файл `tasks_history.json`.

| 🏠 Дом | Убраться в комнате, Помыть посуду |

## Требования для запуска

### Необходимое программное обеспечение:

- **Python 3.6** или выше
- Стандартные библиотеки Python (входят в состав Python):
  - `tkinter` - для графического интерфейса
  - `json` - для работы с файлами
  - `random` - для генерации случайных задач
  - `datetime` - для работы с датой и временем
  - `os` - для работы с файловой системой

Дополнительные библиотеки не требуются.
